--Sobre funções. Normalmente temos que declarar a função: function meuNome(){ console.log("Junior"; } E depois invocar a função para que ela execute. meuNome(); Mas podemos fazer uma função que ao declará-la e defini-la já pode executar. Basta colocar parenteses nela: (function meuNome(){ console.log("Junior"; })(); Assim ela vai ser declarada, definida e invocada. É o que se chama de immediately invoked function.
--Sobre callbacks, é uma função que é passada como argumento para outra função. Geralmente é utilizada em programação baseada eventos. A função principal fica aguardando um evento, então chama uma função. Nas primeiras vezes que vi uma callback eu pensava na necesidade de passar uma função como argumento para outra. Nos meus casos concretos, bastava uma função chamar a outra. Acontece que fazendo assim a função se torna menos dinâmica. Com callback é possível chamar diversas funções. E também é possível em vez de passar uma função definida previamente, é possível passar uma função anônima definida no parâmetro da função principal. Então podemos ter para map: 
    array.map(nomeDaFunçãoCallbal);
    ou
    arry.map((item) => item * 2);
--No primeiro caso uma função já definida é passada para o método map como callback. No segundo caso passamos uma função anônima que é expressa no próprio campo dos argumentos.

Existe um grande problema com o formato de data do javascript. É interessante notar que existe um padrão iso. myDate.toISOString(); ISO 8601.
In JavaScript, the Date() function returns a string representation of the current date and time. The string is formatted according to the ISO 8601 standard, which is a standardized format for representing date and time information. The ISO 8601 format for a date and time string is YYYY-MM-DDTHH:mm:ss.sssZ, where: 
YYYY is the four-digit year.
    MM is the two-digit month (01-12).
    DD is the two-digit day of the month (01-31).
    T is the date-time separator.
    HH is the two-digit hour (00-23).
    mm is the two-digit minute (00-59).
    ss is the two-digit second (00-59).
    sss is the three-digit millisecond (000-999).
    Z is the time zone offset.
Here's an example of how you might use the Date() function in JavaScript: let now = new Date(); console.log(now.toISOString()); // Outputs the current date and time in ISO 8601 format
